Artefak Saya

Project Based – Project UAS Mobile Programming (2024)

Project ini merupakan pengembangan aplikasi berbasis mobile yang dikerjakan sebagai tugas Ujian Akhir Semester pada mata kuliah Mobile Programming dengan menggunakan bahasa pemrograman Python dan framework Flutter. Dalam project ini, saya berperan sebagai pemrogram yang bertanggung jawab menghubungkan desain antarmuka yang dibuat oleh rekan tim dengan API yang telah disediakan oleh dosen, sehingga data dapat ditampilkan secara dinamis dan sesuai dengan kebutuhan aplikasi. Melalui project ini, saya memperoleh pemahaman bahwa pengembangan aplikasi mobile tidak hanya berfokus pada tampilan, tetapi juga pada integrasi sistem, pengelolaan data, serta kerja sama tim. Project ini masih memiliki potensi pengembangan lebih lanjut, seperti penambahan fitur daftar lagu yang dapat diakses secara bebas, dan pengalaman yang diperoleh menjadi bekal penting bagi saya dalam menghadapi pengembangan aplikasi mobile di dunia profesional.

GitHub: https://github.com/AgusDwi90/Project_UAS

Case Based – Sistem Informasi PORSENIJAR Kota Denpasar Berbasis WEB (TA)

Studi kasus ini membahas pengelolaan kegiatan PORSENIJAR Kota Denpasar yang sebelumnya masih dilakukan menggunakan aplikasi umum seperti Google Form dan Google Spreadsheet, sehingga menimbulkan berbagai permasalahan seperti duplikasi data, terhapusnya data, dan terlambatnya penginformasian perolehan medali. Untuk mengatasi hal tersebut, dikembangkan sistem informasi berbasis web dengan menerapkan konsep Role-Based Access Control (RBAC) menggunakan metode pengembangan Prototype karena banyaknya pengguna sistem, sehingga setiap pengguna memiliki hak akses sesuai perannya. Melalui studi kasus ini, saya memahami bahwa keamanan sistem tidak hanya berkaitan dengan perlindungan data, tetapi juga pengaturan hak akses yang tepat serta perancangan sistem yang terstruktur. Pengalaman ini memperkuat pemahaman saya mengenai pentingnya analisis kebutuhan, perancangan sistem, dan penerapan konsep keamanan dalam membangun sistem informasi yang andal dan siap diterapkan di lingkungan organisasi nyata.
